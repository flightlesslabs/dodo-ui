## 📦 Packages

### 🧱 Core

- **[Dodo UI](https://github.com/flightlesslabs/dodo-ui/tree/main/packages/dodo-ui)**  
  The core design system and component library for building consistent Svelte applications.

### 🧩 Addons

- **[dodo-ui-bits](https://github.com/flightlesslabs/dodo-ui/tree/main/packages/dodo-ui-bits)**  
  An extension for Dodo UI that provides advanced, accessible components powered by [Bits UI](https://bits-ui.com/) (Select, Checkbox)

- **[dodo-ui-date](https://github.com/flightlesslabs/dodo-ui/tree/main/packages/dodo-ui-date)**  
  An extension for Dodo UI that provides accessible date and time components powered by by [Bits UI](https://bits-ui.com/) (DatePicker)

- **[dodo-ui-numeric](https://github.com/flightlesslabs/dodo-ui/tree/main/packages/dodo-ui-numeric)**  
  An extension for Dodo UI that provides formatted numeric input components powered by [svelte-number-format](https://github.com/pitis/svelte-number-format).
