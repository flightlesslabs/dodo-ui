# Dodo Ui

Opinionated UI framework for Svelte, based on [Bits UI](https://bits-ui.com/)

[Github](https://github.com/flightlesslabs/dodo-ui) | [npm](https://www.npmjs.com/package/@flightlesslabs/dodo-ui)

## Develop

### Run app

1. `pnpm i`
2. `pnpm dev`

### Build app

1. `pnpm check`
2. `pnpm build`
