# Dodo Ui

An open-source, opinionated UI framework for Svelte.

[Github](https://github.com/flightlesslabs/dodo-ui) | [npm](https://www.npmjs.com/package/@flightlesslabs/dodo-ui)

## Develop

### Run app

1. `pnpm i`
2. `pnpm dev`

### Build app

1. `pnpm check`
2. `pnpm build`
