# @flightlesslabs/dodo-core-attributes

Dodo core utility package providing standardized design tokens and property variations such as color, roundness, and shape.

This package serves as the foundation for the Dodo UI ecosystem, ensuring consistency across components while remaining flexible enough to be used independently in other projects.
