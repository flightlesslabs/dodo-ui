## Install

pnpm

<Source
  dark
  language="bash"
  code={`
pnpm add @flightlesslabs/dodo-ui
`}
/>

npm

<Source
  dark
  language="bash"
  code={`
npm i @flightlesslabs/dodo-ui
`}
/>

## Add css

Add global css in your `+layout.svelte` file

<Source
  dark
  language="js"
  code={`
import '@flightlesslabs/dodo-ui/styles/global.css';
`}
/>
